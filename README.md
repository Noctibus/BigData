# Spoutify

## Description

A simple website to search for a Spotify song and display Spotify statistics.
## Getting Started

### Dependencies

* Any OS (Linux, Windows, Mac)
* Python libraries (flask, pymongo)
* MongoDB installed

### Installing

* Clone the repository

### Executing program

* Run the file serveur.py
```
python serveur.py
```
